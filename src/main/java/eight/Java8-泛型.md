泛型通过在编译时检测到更多错误来增加代码的稳定性。

J2SE 5.0中引入了这种期待已久的增强类型系统的功能，它允许类型或方法对各种类型的对象进行操作，同时提供编译时类型的安全性。 它为集合框架增加了编译时类型的安全性，并消除了繁琐的转换。
